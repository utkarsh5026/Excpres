# Excpres

A Python project managed with Poetry.

## Installation

```bash
poetry install
```

## Usage

```bash
poetry run python -m excpres
```

## Development

### Install development dependencies

```bash
poetry install --with dev
```

### Run tests

```bash
poetry run pytest
```

### Code formatting

```bash
poetry run black .
```

### Linting

```bash
poetry run flake8
```
